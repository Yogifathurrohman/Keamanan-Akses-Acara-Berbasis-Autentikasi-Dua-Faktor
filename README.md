Pembayaran & Verifikasi Acara (GUI Based with 2FA)
Sistem manajemen pendaftaran dan verifikasi peserta acara berbasis desktop menggunakan library Tkinter Python. Aplikasi ini dirancang untuk mensimulasikan alur pendaftaran tiket hingga verifikasi keamanan di lokasi acara menggunakan sistem OTP (Two-Factor Authentication).

Fitur Utama:
1. Pendaftaran Peserta (Multi-Page Interface): Alur kerja 4 halaman yang dinamis (Pendaftaran, Hasil Pembayaran, Verifikasi, dan Riwayat).
2. Object-Oriented Programming (OOP): Menggunakan kelas Peserta dengan konsep Encapsulation (Getter) untuk mengelola data secara terstruktur.
3. Sistem Keamanan 2FA (OTP): Simulasi verifikasi dua langkah menggunakan kode OTP acak untuk memastikan validitas peserta.
4. Blacklist Management: Fitur keamanan untuk memblokir nama-nama tertentu yang masuk dalam daftar hitam.
5. Manajemen Data Real-time: Penyimpanan data peserta dalam array/list dengan batasan kuota maksimal (MAX_PESERTA).
6. Antarmuka Modern: Menggunakan tema clam pada ttk dan skema warna Dark Mode untuk estetika yang profesional.
